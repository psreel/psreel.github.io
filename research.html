Projects

HT-Advance project

July 2023 - Present

This EU funded project involves improving treatment efficacy in hypertension by biomarker-guided personalised decision support. I am Working on the development of a clinically-relevant machine learning tool for the diagnosis of hypertension from clinical multi-omics data.

PICTURES project

Jan 2022 - June 2023

This project is MRC and EPSRC funded and aims to enable international research access to highly heterogeneous, routinely collected, linked clinical images. These images can be used for examining early/pre-clinical diagnosis, disease progression, genotype-phenotype associations, development of risk profiles and classification of disease types

ENS@T-HT project

March 2016 - Dec 2021

This project is an EU-funded Horizon 2020 research and innovation project that will investigate specific types of hypertension and whether it is possible to define specific profiles based on systems biology ”-omics” approaches and subsequently use these to define more personalised approaches to disease management. It includes developing bioinformatics tools/models and advanced machine learning-driven data analytics.

Multi-morbidity Guideline Development Tool

January 2020 - April 2020

This pilot work involved development of an interactive dashboard prototype for Multi-morbidity guideline development. The preliminary graphical user interface was implemented using RShiny software.

Multimodal Medical Image Registration

2011 - 2016

This project focuses on developing a new similarity measure for robust registration of challenging medical images with severe artifacts. It aims to design similarity measure which can efficiently incorporate spatial features with mutual information without incurring high computational overheads for brain & retinal datasets.

Feature Extraction for Enhanced Wheat Quality Inspection

2011

  • This project focused on the classification of wheat grains using different classifiers combined to morphological & colour features.

Autonomous Ground Vehicle for Low Intensity Conflict

2010

The Autonomous Ground Vehicle is to be part of the future combat force to carry out tasks that would otherwise be difficult for a conventional manned vehicle. This project was showcased at Lockheed Martin Center for Innovation, Suffolk, VA, USA at Young Engineers Visitation Program under the Joint Lockheed Martin & Indo-US Science & Technology Forum Academic Program.

Low Cost Outdoor Unmanned Aerial Imaging System

2008 - 2009

This project is a dual controlled (manual as well as autopilot) Deployable Low Cost Outdoor Surveillance system. The aircraft is designed to have wingspan of less than 6 feet capable of carrying autopilot system & useful payload. Real time video algorithm consists of frame by frame mosaicing & video stabilization. Finally object recognition is performed.

Bit Error Rate based First Read Rate Analysis of Low Grade 2D Barcode

2008 - 2009

This project investigated the various factors which impact the robustness of 2D-barcode reading & how can it be improved.

Insertion of Watermark in 3G Video Telephony

2003 - 2005

This project dealt with video watermarking in frequency domain for security purposes. Watermarked videos are compressed at high & low bit rates using H.263 codec & then PSNR based qualitative evaluation was conducted & applied for 3G Video telephony.

Wavelet Based Personnel Identification System (Biometrics)

2000 - 2002

This project designed a fingerprint image encryption & recognition algorithm based on Wavelet Transformation. This algorithm was initially coded in Matlab & then ported as C executable. This Project won Indian National Award for year 2002 .

Spectrum Analyser using Fourier Transformation

1999 - 2000

This project demonstrated the use of Fourier transformation for analysing various signals & plots their magnitude & phase. This project was implemented in C Language.
